# Docker Swarm Operator

> GitOps operator for Docker Swarm.

## Getting started

```bash
swarm_operator --git-repo git@github.com:blesswinsamuel/swarm-stack-example.git --private-key-file ~/.ssh/id_rsa --repo-dir /tmp/swarm-operator-repo
```
